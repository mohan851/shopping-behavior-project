# Shopping Behavior Analysis

This project analyzes customer shopping behavior to understand purchasing patterns,
customer trends, and product preferences.

## Project Overview
The goal of this project is to gain insights into customer shopping data by performing
Exploratory Data Analysis (EDA) and creating visual dashboards for better decision-making.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
- Power BI

## Work Done
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA) using Jupyter Notebook
- Identification of shopping patterns and trends
- Data visualization using Power BI dashboards

## Results
- Identified key customer purchasing behaviors
- Analyzed popular products and sales trends
- Visual insights presented using Power BI

## How to Run
1. Open the Jupyter Notebook to view EDA
2. Open the Power BI file to explore visual dashboards

## Future Scope
- Build a recommendation system
- Add predictive models for customer behavior
- Deploy insights using a web application
